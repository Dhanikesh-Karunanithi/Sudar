# Sudar Screenshots

Add 2–4 screenshots here to show that Sudar is building, not just planning. Use the filenames below so the main README can reference them.

## Suggested screenshots

| File | Description |
|------|-------------|
| `learn-dashboard.png` | Learn dashboard — personalized home (streak, next best action, upcoming deadlines, activity chart). |

## How to add

1. Run Learn at http://localhost:3001 and Studio at http://localhost:3000.
2. Capture full-window or key area (e.g. 1280×720 or 1920×1080).
3. Save as PNG in this folder with the exact names above.
4. The main [README.md](../../README.md) already links to this folder; screenshots will appear when present.
