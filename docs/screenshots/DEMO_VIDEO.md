# Sudar Memory Demo Video (1–2 min)

Record a short demo to show that Sudar’s tutor **remembers** the learner across sessions. Use this for pitches, README, or social.

## Goal

Show in 1–2 minutes:

1. **Learner context** — Open Learn, show dashboard (streak, next best action).
2. **Course + Sudar** — Open a course, open the Sudar chat, ask something about the content (e.g. “Explain the key idea of this module”).
3. **Memory** — Show that Sudar can reference the learner (e.g. “My Memory” page or a tutor reply that refers to past preferences/struggles).
4. **Optional** — Quick shot of Studio: path assignment or Compliance view.

## Steps

1. **Learn** (http://localhost:3001): Log in as a learner who has done onboarding and has some history.
2. **Dashboard**: Show home with streak, progress, “Sudar recommends” or “Upcoming deadlines”.
3. **Course**: Go to a course → Learn → open a module. Open the floating Sudar chat (bottom-right).
4. **Ask Sudar**: Type a question about the module or ask “What do you remember about me?” — show a response that uses memory/context.
5. **My Memory** (optional): Go to Memory page and show the editable summary Sudar uses.
6. **Studio** (optional): Switch to Studio, show Compliance or Path editor with assigned learners.

## Output

- **Format**: MP4 or WebM, 1–2 minutes, 1280×720 or 1920×1080.
- **Where to put it**: e.g. `docs/screenshots/sudar-memory-demo.mp4` or host on YouTube/Vimeo and link from README.
- **README**: Add a line in the main README under “Screenshots” like:  
  *“Watch a 1–2 min demo of Sudar’s memory: [link].”*
